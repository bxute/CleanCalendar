# CleanCalendar

## A lightweight Android library for calendar.

#### Features:
 - Event indicator
 - Selectable dates
 - Formatting of day names as "Monday" | "Mon" | "M"
 - Customizable color for each items
 - Callbacks for date/month change
 - Adapter for setting events for a particular day.
